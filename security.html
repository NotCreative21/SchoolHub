Security Bot

What is the security bot?
Security Bot is a simple discord bot that can perform some task to prevent spam in discords and also just provide some useful utilities.

Discord bot features

features | description
Purge Purge messages from channels
Ping Measure bot response time
Nuke Nuke channel and clone permissions to new channel
